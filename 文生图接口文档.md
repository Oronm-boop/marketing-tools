# ComfyUI 文生图接口文档

接口基础地址: `http://192.168.0.122:8188`

## 1. 调用流程

1. 调用 `POST /prompt` 下发 ComfyUI 工作流任务。
2. 从返回值中获取 `prompt_id`。
3. 调用 `GET /history/{prompt_id}` 查询任务执行结果。
4. 从 `outputs` 中取出生成文件信息，例如 `filename`、`subfolder`、`type`。
5. 调用 `GET /view` 预览或下载生成图片。

## 2. 接口总览

| 方法 | 路径 | 用途 |
| --- | --- | --- |
| `POST` | `/prompt` | 下发绘图任务，返回任务 ID |
| `GET` | `/prompt` | 获取服务器当前剩余任务队列数量 |
| `GET` | `/history/{prompt_id}` | 按任务 ID 获取历史任务数据 |
| `GET` | `/history` | 获取历史任务列表 |
| `GET` | `/view` | 预览或读取图片文件 |
| `POST` | `/upload/image` | 上传图片 |
| `POST` | `/upload/mask` | 上传蒙版图片 |
| `POST` | `/interrupt` | 取消当前任务 |
| `GET` | `/queue` | 获取运行中和等待中的队列 |
| `POST` | `/queue` | 清空队列或删除指定队列任务 |
| `GET` | `/object_info` | 获取所有节点及参数信息 |
| `GET` | `/object_info/{node_class}` | 获取指定节点类型的参数信息 |
| `GET` | `/system_stats` | 获取系统和显卡统计信息 |
| `POST` | `/view_metadata/{folder_name}` | 获取文件夹元数据，线上文档标记为开发中 |

## 3. 核心接口: 下发文生图任务

### `POST /prompt`

完整 URL:

```text
http://192.168.0.122:8188/prompt
```

说明: 下发绘图任务。该接口只提交任务，成功后返回 `prompt_id`，后续需要通过 `/history/{prompt_id}` 查询结果。

请求头:

```http
Content-Type: application/json
```

请求体字段:

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `client_id` | `string` | 是 | 客户端 ID，可使用 UUID |
| `prompt` | `object` | 是 | ComfyUI 工作流 JSON，节点 ID 之间通过数组引用连接 |

常用可替换参数:

| 位置 | 字段 | 说明 |
| --- | --- | --- |
| `prompt.4.inputs.ckpt_name` | 模型文件名 | 使用当前 ComfyUI 可用的 checkpoint |
| `prompt.5.inputs.width` | 图片宽度 | 示例为 `512` |
| `prompt.5.inputs.height` | 图片高度 | 示例为 `512` |
| `prompt.5.inputs.batch_size` | 批量数量 | 示例为 `1` |
| `prompt.6.inputs.text` | 正向提示词 | 文生图主提示词 |
| `prompt.7.inputs.text` | 负向提示词 | 排除内容或质量约束 |
| `prompt.3.inputs.seed` | 随机种子 | 固定种子可复现结果 |
| `prompt.3.inputs.steps` | 采样步数 | 示例为 `30` |
| `prompt.3.inputs.cfg` | CFG | 示例为 `8` |
| `prompt.3.inputs.sampler_name` | 采样器 | 示例为 `dpmpp_2m` |
| `prompt.3.inputs.scheduler` | 调度器 | 示例为 `karras` |
| `prompt.9.inputs.filename_prefix` | 输出文件名前缀 | 示例为 `ComfyUI` |

## 4. 查询任务结果

### `GET /history/{prompt_id}`

完整 URL 示例:

```text
http://192.168.0.122:8188/history/3604be44-eb6b-4d54-b82d-62d31a6c0b36
```

说明: 根据 `prompt_id` 获取历史任务数据，通常用于获取生成图片的文件名。

路径参数:

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `prompt_id` | `string` | 是 | `POST /prompt` 返回的任务 ID |

成功响应结构示例:

```json
{
  "39d61fa4-58a4-4f61-a547-caab0f4c3a53": {
    "prompt": [],
    "outputs": {
      "9": {
        "images": [
          {
            "filename": "ComfyUI_00138_.png",
            "subfolder": "",
            "type": "output"
          }
        ]
      }
    },
    "status": {
      "status_str": "success",
      "completed": true
    }
  }
}
```

取图方式:

1. 找到 `outputs` 中保存图片节点的 `images` 数组。
2. 取出 `filename`、`subfolder`、`type`。
3. 将这些字段作为 `/view` 的查询参数。

### `GET /history`

完整 URL:

```text
http://192.168.0.122:8188/history
```

说明: 获取历史任务列表。返回结构与 `/history/{prompt_id}` 类似，但会包含多个任务 ID。

## 5. 预览或读取图片

### `GET /view`

完整 URL:

```text
http://192.168.0.122:8188/view
```

说明: 图片在线预览接口。上传图像、生成图像、蒙版图像均通过该接口预览或读取。

Query 参数:

| 参数 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `filename` | `string` | 是 | `ComfyUI_00138_.png` | 图片名称 |
| `type` | `string` | 否 | `output` | 图片存放位置，常见值为 `input`、`output` |
| `subfolder` | `string` | 否 | 空字符串 | 子文件夹，没有可不填 |
| `preview` | `string` | 否 | `WEBP` | 预览格式 |
| `channel` | `string` | 否 | `rgb` | 颜色通道 |

示例:

```text
http://192.168.0.122:8188/view?filename=ComfyUI_00138_.png&type=output&subfolder=&preview=WEBP&channel=rgb
```

响应: 图片二进制数据。

## 6. 队列相关接口

### `GET /prompt`

完整 URL:

```text
http://192.168.0.122:8188/prompt
```

说明: 获取服务器当前剩余任务队列数量。

成功响应示例:

```json
{
  "exec_info": {
    "queue_remaining": 0
  }
}
```

### `GET /queue`

完整 URL:

```text
http://192.168.0.122:8188/queue
```

说明: 获取当前运行中和等待中的任务队列。

成功响应示例:

```json
{
  "queue_running": [],
  "queue_pending": []
}
```

### `POST /queue` 清空队列

完整 URL:

```text
http://192.168.0.122:8188/queue
```

请求体:

```json
{
  "clear": true
}
```

说明: 清除队列。线上文档说明无返回信息则为成功。

### `POST /queue` 删除指定任务

完整 URL:

```text
http://192.168.0.122:8188/queue
```

请求体:

```json
{
  "delete": "c8e286d2-9660-46a3-b3e5-b763b6654386"
}
```

字段说明:

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `delete` | `string` | 是 | 任务下发时返回的 `prompt_id` |

说明: 删除指定队列任务。线上文档说明无返回信息则为成功。

## 7. 上传接口

### `POST /upload/image`

完整 URL:

```text
http://192.168.0.122:8188/upload/image
```

说明: 上传图片。

请求类型:

```http
Content-Type: multipart/form-data
```

表单字段:

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `image` | `file` | 是 | 需要上传的图片文件 |

curl 示例:

```bash
curl -X POST "http://192.168.0.122:8188/upload/image" \
  -F "image=@D:/images/input.png"
```

成功响应示例:

```json
{
  "name": "0e9f-hiixpup5792613.jpg",
  "subfolder": "",
  "type": "input"
}
```

### `POST /upload/mask`

完整 URL:

```text
http://192.168.0.122:8188/upload/mask
```

说明: 上传蒙版图片，一般用于局部重绘。

请求类型:

```http
Content-Type: multipart/form-data
```

表单字段:

| 字段 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `image` | `file` | 是 | `mask.png` | 蒙版图片 |
| `type` | `string` | 否 | `input` | 文件类型 |
| `subfolder` | `string` | 否 | `clipspace` | 子目录 |
| `original_ref` | `string` | 否 | `{"filename":"下载.png","type":"input","subfolder":"clipspace"}` | 原图引用 |

成功响应示例:

```json
{
  "name": "640.png",
  "subfolder": "clipspace",
  "type": "input"
}
```

http://192.168.0.122:8188/interrupt

## 9. 节点信息接口

### `GET /object_info`

完整 URL:

```text
http://192.168.0.122:8188/object_info
```

说明: 获取系统中所有组件以及可用参数。

响应说明:

返回对象以节点类型为 key，例如 `KSampler`、`CheckpointLoaderSimple`、`CLIPTextEncode` 等。每个节点通常包含:

| 字段 | 说明 |
| --- | --- |
| `input.required` | 必填输入参数 |
| `input.optional` | 可选输入参数 |
| `output` | 输出类型 |
| `output_name` | 输出名称 |
| `name` | 节点类名 |
| `display_name` | 显示名称 |
| `category` | 节点分类 |
| `output_node` | 是否为输出节点 |

### `GET /object_info/{node_class}`

完整 URL 示例:

```text
http://192.168.0.122:8188/object_info/KSampler
```

说明: 根据组件名称获取该组件的参数。

路径参数:

| 参数 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `node_class` | `string` | 是 | `KSampler` | ComfyUI 节点类名 |

`KSampler` 常见参数:

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `model` | `MODEL` | 模型输入 |
| `seed` | `INT` | 随机种子 |
| `steps` | `INT` | 采样步数 |
| `cfg` | `FLOAT` | CFG 强度 |
| `sampler_name` | `string` | 采样器名称 |
| `scheduler` | `string` | 调度器名称 |
| `positive` | `CONDITIONING` | 正向条件 |
| `negative` | `CONDITIONING` | 负向条件 |
| `latent_image` | `LATENT` | Latent 输入 |
| `denoise` | `FLOAT` | 去噪强度 |

线上示例中的采样器包括:

```text
euler, euler_ancestral, heun, heunpp2, dpm_2, dpm_2_ancestral, lms, dpm_fast, dpm_adaptive, dpmpp_2s_ancestral, dpmpp_sde, dpmpp_sde_gpu, dpmpp_2m, dpmpp_2m_sde, dpmpp_2m_sde_gpu, dpmpp_3m_sde, dpmpp_3m_sde_gpu, ddpm, lcm, ddim, uni_pc, uni_pc_bh2
```

线上示例中的调度器包括:

```text
normal, karras, exponential, sgm_uniform, simple, ddim_uniform
```
